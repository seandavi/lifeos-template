## What This File Is

**now.md** answers: *what am I actually working on, and where does each thing stand?*

It is **not**:
- A task list → use `inbox.md`
- Annual strategy → use `journal/YYYY/goals.md`
- Project details → use `projects/` files
- Completed work → use `completed.md`

**Update trigger:** After any significant session, or at minimum weekly via `/weekly-review`. If it hasn't been touched in 10 days, it's lying.
